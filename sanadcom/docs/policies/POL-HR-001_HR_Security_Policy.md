## 1. Purpose | الغرض

### English
Ensure security responsibilities are understood before employment, during employment, and after termination in compliance with ISO 27001 A.6 (People controls) and NCA ECC-HR requirements.

### العربية
ضمان فهم المسؤوليات الأمنية قبل التوظيف وأثناء التوظيف وبعد الإنهاء بما يتوافق مع ISO 27001 A.6 (ضوابط الأفراد) ومتطلبات NCA ECC-HR.

## 2. Pre-Employment Screening | الفحص قبل التوظيف

### English

**Background Checks (Required):**

✅ **Identity Verification**: Government-issued ID verification  
✅ **Employment History**: Last 5 years verified  
✅ **Education Verification**: Degrees confirmed  
✅ **Criminal Record Check**: Saudi Arabia + country of residence  
✅ **Credit Check**: For financial/privileged access roles  
✅ **Reference Checks**: Minimum 2 professional references  

**Additional Screening for Sensitive Roles:**
- Security clearance for CISO, Security Team
- Enhanced background check for DBA, System Administrators
- Financial background for accounting/finance roles
- Continuous monitoring for privileged users

**Third-Party Personnel:**
- Same screening requirements apply
- Supplier responsible for background checks
- SICO verifies completion before access granted

### العربية

**فحوصات الخلفية (مطلوبة):**

✅ **التحقق من الهوية**: التحقق من الهوية الصادرة عن الحكومة  
✅ **تاريخ التوظيف**: التحقق من آخر 5 سنوات  
✅ **التحقق من التعليم**: تأكيد الشهادات  
✅ **فحص السجل الجنائي**: المملكة العربية السعودية + بلد الإقامة  
✅ **فحص الائتمان**: للأدوار المالية/الوصول المميز  
✅ **فحص المراجع**: الحد الأدنى 2 مراجع مهنية  

**الفحص الإضافي للأدوار الحساسة:**
- تصريح أمني لمدير أمن المعلومات، فريق الأمن
- فحص خلفية معزز لمسؤول قاعدة البيانات، مسؤولي النظام
- خلفية مالية لأدوار المحاسبة/المالية
- المراقبة المستمرة للمستخدمين المميزين

**موظفو الطرف الثالث:**
- تنطبق نفس متطلبات الفحص
- المورد مسؤول عن فحوصات الخلفية
- تتحقق سيكو من الاكتمال قبل منح الوصول

## 3. Employment Agreements | اتفاقيات التوظيف

### English

**Mandatory Clauses:**

1. **Confidentiality Agreement (NDA)**
   - Protect company confidential information
   - Includes customer data, trade secrets, IP
   - Survives employment termination

2. **Acceptable Use Policy Acceptance**
   - Computer and network usage guidelines
   - Monitoring and privacy expectations
   - Consequences of policy violations

3. **Security Responsibilities**
   - Comply with security policies
   - Report security incidents
   - Protect credentials and access rights
   - Complete mandatory security training

4. **Intellectual Property Assignment**
   - Work product belongs to SICO
   - No unauthorized use of company resources for personal projects

5. **Return of Property**
   - Return all company equipment upon termination
   - Includes laptops, phones, access cards, documents

**Annual Re-Acknowledgement:**
- All employees re-acknowledge policies annually
- Changes communicated and acceptance logged
- Non-compliance triggers HR review

### العربية

**البنود الإلزامية:**

1. **اتفاقية السرية (NDA)**
   - حماية المعلومات السرية للشركة
   - تشمل بيانات العملاء، الأسرار التجارية، الملكية الفكرية
   - تستمر بعد إنهاء التوظيف

2. **قبول سياسة الاستخدام المقبول**
   - إرشادات استخدام الكمبيوتر والشبكة
   - توقعات المراقبة والخصوصية
   - عواقب انتهاكات السياسة

3. **المسؤوليات الأمنية**
   - الامتثال للسياسات الأمنية
   - الإبلاغ عن الحوادث الأمنية
   - حماية بيانات الاعتماد وحقوق الوصول
   - إكمال التدريب الأمني الإلزامي

4. **تعيين الملكية الفكرية**
   - منتج العمل ملك لسيكو
   - لا استخدام غير مصرح به لموارد الشركة للمشاريع الشخصية

5. **إرجاع الممتلكات**
   - إرجاع جميع معدات الشركة عند الإنهاء
   - تشمل أجهزة الكمبيوتر المحمولة والهواتف وبطاقات الوصول والوثائق

**إعادة الإقرار السنوي:**
- يعيد جميع الموظفين الإقرار بالسياسات سنويًا
- يتم الإبلاغ عن التغييرات وتسجيل القبول
- عدم الامتثال يؤدي إلى مراجعة الموارد البشرية

## 4. Termination Process | عملية الإنهاء

### English

**Offboarding Checklist:**

**HR Responsibilities (T-0 hours):**
1. Notify IT Security within 2 hours of termination decision
2. Coordinate equipment return
3. Conduct exit interview
4. Remind of ongoing confidentiality obligations

**IT Security Responsibilities (Immediate):**
1. ✅ Disable all user accounts (AD, email, VPN, applications)
2. ✅ Revoke remote access (VPN, cloud services)
3. ✅ Deactivate physical access cards
4. ✅ Remove from distribution lists and shared folders
5. ✅ Retrieve laptops, phones, tokens, badges
6. ✅ Transfer data ownership to manager
7. ✅ Document all actions in offboarding ticket

**Manager Responsibilities (Within 48 hours):**
1. Review user's files and data
2. Identify business-critical information
3. Transfer ownership as appropriate
4. Delete personal/non-business files

**Post-Termination (30 days):**
- Account deleted permanently
- Backup data retained per retention policy
- Final offboarding report to HR and CISO

### العربية

**قائمة تحقق الإنهاء:**

**مسؤوليات الموارد البشرية (T-0 ساعات):**
1. إخطار أمن تقنية المعلومات خلال ساعتين من قرار الإنهاء
2. تنسيق إرجاع المعدات
3. إجراء مقابلة الخروج
4. التذكير بالتزامات السرية المستمرة

**مسؤوليات أمن تقنية المعلومات (فوري):**
1. ✅ تعطيل جميع حسابات المستخدم (AD، البريد الإلكتروني، VPN، التطبيقات)
2. ✅ إلغاء الوصول عن بُعد (VPN، الخدمات السحابية)
3. ✅ إلغاء تنشيط بطاقات الوصول المادي
4. ✅ الإزالة من قوائم التوزيع والمجلدات المشتركة
5. ✅ استرجاع أجهزة الكمبيوتر المحمولة والهواتف والرموز والشارات
6. ✅ نقل ملكية البيانات إلى المدير
7. ✅ توثيق جميع الإجراءات في تذكرة الإنهاء

**مسؤوليات المدير (خلال 48 ساعة):**
1. مراجعة ملفات وبيانات المستخدم
2. تحديد المعلومات الحيوية للأعمال
3. نقل الملكية حسب الاقتضاء
4. حذف الملفات الشخصية/غير التجارية

**بعد الإنهاء (30 يومًا):**
- حذف الحساب نهائيًا
- الاحتفاظ ببيانات النسخ الاحتياطي وفقًا لسياسة الاحتفاظ
- تقرير الإنهاء النهائي للموارد البشرية ومدير أمن المعلومات
