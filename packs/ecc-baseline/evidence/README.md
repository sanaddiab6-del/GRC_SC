
## Overview
This directory contains evidence collection templates for NCA Essential Cybersecurity Controls (ECC). Each template is designed to facilitate audit readiness and compliance verification.

## Evidence Categories

### 1. Governance Evidence
**Controls**: ECC-GV-1, ECC-GV-2, ECC-GV-3

**Templates**:
- `cybersecurity_policy_template.docx` - Master cybersecurity policy
- `risk_register_template.xlsx` - Risk assessment tracking
- `asset_inventory_template.xlsx` - Information asset register
- `board_approval_form.docx` - Management approval documentation

### 2. Access Control Evidence
**Controls**: ECC-IS-1, ECC-IS-3

**Templates**:
- `access_control_policy.docx` - Access control procedures
- `user_access_matrix.xlsx` - Role-based access matrix
- `access_request_form.docx` - Access request and approval
- `access_review_checklist.xlsx` - Quarterly access review
- `mfa_implementation_report.docx` - MFA deployment evidence

### 3. Cryptography Evidence

- `key_management_procedure.docx` - Key lifecycle management

### 4. Network Security Evidence

## Using Evidence Templates

### Step 1: Select Appropriate Template
Choose the template that corresponds to the control you're implementing.

### Step 2: Customize for Your Organization
- Replace [Organization Name] with your company name
- Add specific technical details
- Include dates and responsible parties
- Adjust based on your environment

### Step 3: Collect Supporting Evidence
Gather screenshots, configurations, logs, or other artifacts that demonstrate control implementation.

### Step 4: Store in Evidence Repository
Organize evidence in your document management system for audit retrieval.

## Evidence Naming Convention
```
[Control ID]_[Evidence Type]_[YYYY-MM-DD].[extension]

Examples:
- ECC-GV-1_Cybersecurity_Policy_2024-01-15.docx
- ECC-IS-1_Access_Review_2024-Q1.xlsx
- ECC-IS-5_Incident_Log_2024-02.xlsx
```

## Audit Readiness Checklist
For each control, ensure you have:
- [ ] Policy or procedure document
- [ ] Implementation evidence (screenshots, configs)
- [ ] Monitoring/review records (logs, reports)
- [ ] Training records (if applicable)
- [ ] Management approval (where required)

## Bilingual Support
All templates are available in:
- English (default)
- Arabic (العربية) - Templates with `_ar` suffix

## Template Maintenance
- Review templates quarterly
- Update based on NCA guidance changes
- Version control all templates
- Archive superseded versions
