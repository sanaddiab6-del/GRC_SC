# Audit Readiness Checklist

## Purpose

This checklist verifies readiness for ISO 27001 Stage 1 and Stage 2 audits, and regulatory reviews (NCA ECC/CCC, PDPL).

---

## A. ISMS Documentation

- [x] ISMS Manual (Clauses 4-10)
- [x] Statement of Applicability (SOA)
- [x] Risk Assessment and Treatment Plan
- [x] 45 information security policies published
- [x] Procedures and SOPs documented (100+)
- [x] Policy acknowledgement tracking enabled

## B. Risk Management

- [x] Risk assessment completed within last 12 months
- [x] Risk register maintained and reviewed quarterly
- [x] Residual risks approved by management
- [x] Risk treatment actions implemented

## C. Access Control

- [x] RBAC implemented across systems
- [x] MFA enforced for all accounts
- [x] Quarterly access reviews completed
- [x] Privileged access managed (PAM)

## D. Incident Management

- [x] CSIRT established with clear roles
- [x] Incident response plan tested (tabletop exercises)
- [x] Breach notification process documented (PDPL 72-hour)
- [x] Incident logs maintained and reviewed

## E. Business Continuity

- [x] Business Continuity Plan (BCP) documented
- [x] Disaster Recovery Plan (DRP) documented
- [x] Backups tested monthly
- [x] DR failover test performed quarterly

## F. Training and Awareness

- [x] Mandatory training modules completed (1-5)
- [x] Role-based training assigned
- [x] Phishing simulations conducted quarterly
- [x] Training records retained (7 years)

## G. Audit Program

- [x] Internal audit plan established (annual)
- [x] Internal audits completed for current year
- [x] Corrective actions tracked and closed
- [x] Management review completed

## H. Evidence Library

- [x] Evidence mapping for ISO 27001 controls
- [x] Evidence mapping for NCA ECC/CCC
- [x] Evidence mapping for PDPL and SDAIA AI
- [ ] Evidence collection scripts automated (in progress)
