## 🚀 كيفية الاستخدام / How to Use

### 1. عرض الصفحات العربية
```bash
# فتح مكتبة الضوابط العربية
Start-Process "CONTROL_LIBRARY_AR.html"

# فتح العرض التوضيحي العربي
Start-Process "DEMO_AR.html"
```

### 2. تشغيل الواجهة العربية (Next.js)
```bash
# الانتقال إلى المجلد
cd src/frontend

# تشغيل التطبيق
npm run dev

# فتح المتصفح على
# http://localhost:3000/ar
# (العربية افتراضياً، ستحول تلقائياً من / إلى /ar)
```

### 3. التبديل بين اللغات
- استخدم زر "English" في الواجهة للتبديل من العربية للإنجليزية
- استخدم زر "العربية" للتبديل من الإنجليزية للعربية
- الروابط المباشرة:
  - العربية: `http://localhost:3000/ar`
  - الإنجليزية: `http://localhost:3000/en`
