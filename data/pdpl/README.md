# PDPL Compliance Materials

## Personal Data Protection Law (PDPL) - Saudi Arabia

This directory contains materials for compliance with Saudi Arabia's Personal Data Protection Law (PDPL).

## Overview

PDPL establishes requirements for:
- Data processing activities
- Data subject rights
- Breach notification
- Data retention
- Privacy governance

## Directory Contents

## Key Dates & Deadlines

### Breach Notification Timelines
- **Internal notification**: Within 24 hours of discovery
- **Authority notification**: Within 72 hours (for serious breaches)
- **Data subject notification**: Without undue delay (for high-risk breaches)

### DSAR Response Timeline
- **Response deadline**: 30 days from receipt
- **Extension allowed**: Up to 60 days with justification
