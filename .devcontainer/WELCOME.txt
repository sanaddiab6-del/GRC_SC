📝 After Setup, Start Services:
────────────────────────────────────────────────────────────────────────

1. Start Docker services:
   cd deployment && docker-compose up -d

2. Start Backend API:
   cd src/backend && uvicorn main:app --reload --host 0.0.0.0

3. Start Frontend (optional):
   cd src/frontend && npm run dev
